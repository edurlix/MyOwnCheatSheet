HTML Tables

Creating Table

before displaying data, we must first create the table that will contain the data by using table < table >

< table >

< /table >

Table Rows

the table is already predefined for you, meaning that it contains the rows, columns, and cells that will hold data. In HTML, all of these components must be created. the first step in entering data into the table is to add rows using the table row element: < tr >

< table >

< tr >

< /tr >

< tr >

< /tr >

< /table >

Table Data

Rows aren't sufficient to add data to a table. Each cell element must also be defined. In HTML you can add data using the table data element: < td >

< table >

< tr >

< td > 81 < /td >

< td > 73 < /td >

< /tr >

< /table >

81 73

Table Headings

to add titles to rows and columns, you can use the table headind element: < th >

the table heading element is used just like a table data element, except with a relevant title just like table data, a table heading must be placed within a table row.

< table >

< tr >

< th > < /th >

< th scope="col"> Saturday < /th >

< th scope="col"> Sunday < /th >

< /tr >

< tr >

< th scope="row"> Temperature< /th >

< td > 73 < /td >

< td > 81 < /td >

< /tr >

< /table >

Saturday Sunday
Temperature 73 81

Table Borders

the tables you've created have been a little difficult to read because they have no borders. In order versions of HTML, a border, could be added to a table using the border attribute and setting it equal to an integer. This integer would represent the thickness of the better.

<table border ="1">

< tr >

< td > 73 < /td >

< td > 81 < /td >

< /tr >

< /table >

73 81

other way to add border to a table is using css to style html document

table,td {

   border: 1px solid black;

}

Spanning Columns

what if the table contain data that spans multiple columns?

data can span columns using the colspan attribute. the attribute accepts an integer (greater than or equal to 1) to denote the number of columns it spans accross

< table >

< tr >

< th > Monday < /th >

< th > Tuesday < /th >

< th > Wednesday < /th >

< /tr >

< tr >

<td colspan="2"> Out of Town < /td >

<td > Back in Town < /td >

</tr>

< /table >

Monday Tuesday Wednesday
Out of Town Back in Town

Spanning Rows

Data can also span multiple rows using the rowspan attribute

< table >

  < tr >

    < th >< /th >

    < th > Saturday < /th >

    < th > Sunday < /th >

  < /tr >

  < tr >

    < th > Morning < /th >

     < td rowspan="2"> Work < /td >

     < td rowspan="3"> Relax < /td >

  < /tr >

  < tr >

    < th > Afternoon < /th >

  < /tr >

  < tr >

    < th > Evening < /th >

     < td> Dinner < /td >

  < /tr >

< /table >

Saturday Sunday
Morning Work Relax
Afternoon
Evening Dinner

Table Body

A table can grow to contain a lot of data and become very long, when this happens, the table can be sectioned off so that it is easier to manage. Long tables can be secitoned off using the table body element: <tbody>.

the <tbody> element should contain all of the table's data, excluding the table headings

< table >

  < tbody >

   < tr >

     < th >< /th >

     < th > Saturday < /th >

     < th > Sunday < /th >

   < tr >

     < th > Morning < /th >

   < /tr >

   < tr >

     < th > Evening < /th >

  < tbody >

< /table >
